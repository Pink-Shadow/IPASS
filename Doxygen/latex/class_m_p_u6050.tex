\doxysubsection{Detailed Description}
This class allows you to read and write to the \mbox{\hyperlink{class_m_p_u6050}{MPU6050}} chip. 

\mbox{\hyperlink{class_m_p_u6050}{MPU6050}} acceleration, gyroscope and temperature sensor.

This is a class for the \mbox{\hyperlink{class_m_p_u6050}{MPU6050}}. The \mbox{\hyperlink{class_m_p_u6050}{MPU6050}} is a chip that can measure the gravity acceleration on the X, Y and Z axis, the temperature of the sensor, aswell as a gyroscope. The operating voltage is around 2.\+4V -\/ 3.\+5V (for the exact operating voltage, please refer to the datasheet\+: \href{https://invensense.tdk.com/wp-content/uploads/2015/02/MPU-6000-Datasheet1.pdf}{\texttt{ https\+://invensense.\+tdk.\+com/wp-\/content/uploads/2015/02/\+MPU-\/6000-\/\+Datasheet1.\+pdf}}

The connections to the chip are as followed\+:
\begin{DoxyItemize}
\item VCC\+: 3.\+3V
\item GND\+: ground
\item SCL\+: GPIO pin / scl pin (on the arduino)
\item SDA\+: GPIO pin / sda pin (on the arduino)
\item XDA\+: used to add third party sensors like a magnetor meter
\item XCL\+: used to add third party sensors like a magnetor meter
\item AD0\+: is used to determine the chips I2C address. AD0 to ground\+: 0x68. AD0 to VCC\+: 0x69.
\item INT\+: is used to trigger interrupts.
\end{DoxyItemize}
